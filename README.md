
# Focus-Based Hamburger Menu

## Project Description

This project is a simple responsive hamburger menu created using HTML and CSS. In mobile view, the menu is hidden by default and becomes visible when the hamburger button is focused using the `:focus` property.

No JavaScript is used in this project.

## Technologies Used

* HTML5
* CSS3
* Media Queries
* :focus

## What I Learned

* How to create a responsive navigation menu
* Using `:focus` in CSS
* Handling menu visibility without JavaScript
* Improving basic accessibility using button element

## How to Run

* Download or clone the project.
* Keep `index.html` and `style.css` in the same folder.
* Open `index.html` in your browser.
